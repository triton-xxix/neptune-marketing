## Application

### For Logo Design
Use these exact colors:
- **Primary:** `#0B0F1C` (background)
- **Text:** `#F4F7FB` (headlines)
- **Accent:** `#2EC3E5` (highlights)
- **Secondary text:** `#A9B3C5` (taglines)

### For Print Materials
- Dark backgrounds: Use `#0B0F1C`
- Light backgrounds: White with navy text `#0B0F1C`
- Accent color: Cyan `#2EC3E5` for CTAs
